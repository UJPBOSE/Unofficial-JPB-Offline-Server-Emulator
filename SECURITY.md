# Security and rights-holder contact

This repository is **server emulator source code only**. It is not intended to contain APKs, caches, or other copyrighted game assets.

## Reporting a vulnerability in this project’s code

If you find a security issue in the emulator, launcher, or helper scripts themselves (for example unsafe local file handling), please open a **GitHub Issue** with enough detail to reproduce it. Do not include account tokens or personal save dumps you do not mean to share.

This project is for **local / LAN** use. It is not a public online service.

## Rights holders — DMCA / content removal

If you are a rights holder and believe this repository should be removed:

1. Use **GitHub’s official DMCA process** (not a private email to the maintainer):  
   - Policy: https://docs.github.com/en/site-policy/content-removal-policies/dmca-takedown-policy  
   - Notice form: https://github.com/contact/dmca  
2. The maintainer agrees in advance to comply with a valid GitHub removal of this repository.
3. If GitHub removes this repository, the maintainer will **not** re-upload it to GitHub or stand up an alternate public mirror to evade that removal.

Do not open ordinary Issues asking for game files, APKs, or download links — those requests are out of scope and will be closed.
